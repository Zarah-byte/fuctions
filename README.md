
# Punchy (Functions)
Track your goals with a playful, customizable punch-card generator!

Punchy helps you create a personal goal card with custom text, colors, and icons, then print it as a compact physical tracker. The flow guides you section by section so you can quickly build a card and start tracking right away.

## Features
- Customize your punch cards
- Section-by-section card setup flow
- Goal and reward text input
- Custom punch count selection
- Goal icon selection
- Goal text color + card background color customization
- Live preview while editing
- Print-ready output

## Usage
1. Click **Get Started** on the cover screen.
2. Fill in your goal and reward text.
3. Choose punch count, icon, goal color, and card color.
4. Review the generated card and print it.
5. Use **Make a New Card** to reset and start again.
